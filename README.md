# 🚀 NovaFleet

TransitOps NovaFleet is a next-generation logistics and fleet management platform designed for speed, security, and scalability.

## Core Features
- 🗺️ **Live Map Integration**: Interactive route planning using Google Maps.
- 🔐 **Enterprise Security**: Built-in SSO and automated security scanning with Trivy.
- 📊 **Driver & Vehicle Telematics**: Keep track of driver status, vehicle health, and cargo weights in real-time.
- ✨ **Cosmic UI**: A premium, modern interface optimized for dark mode.

## Tech Stack
- Next.js (App Router)
- React & Framer Motion
- Prisma & PostgreSQL
- NextAuth.js
- Google Maps API
